# 🎵 Spotify Clone

A simple Spotify web player clone built using HTML and CSS.  
This project recreates the basic UI of Spotify including the sidebar, music cards, and music player section.

---

## 🚀 Features

- 🎧 Spotify inspired UI
- 📚 Sidebar navigation (Home, Search, Library)
- 🖼 Music album cards
- ▶️ Music player section
- 🎨 Clean and responsive design
- ⚡ Built with pure HTML and CSS

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts

---

## 💡 What I Learned

- Building UI using HTML & CSS
- Layout design using Flexbox
- Organizing assets and project structure
- Creating a real-world website clone
